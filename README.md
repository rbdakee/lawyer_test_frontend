# Lawyer Test Frontend

Next.js фронтенд для системы подготовки к экзаменам для юристов.

## Установка

```bash
npm install
# или
yarn install
```

## Запуск

```bash
npm run dev
# или
yarn dev
```

Приложение будет доступно по адресу: http://localhost:3000

## Режимы

- **Демо режим** (`/demo`) - показывает правильность ответов сразу после выбора
- **Экзамен** (`/exam`) - режим настоящего экзамена с таймером

## Технологии

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
